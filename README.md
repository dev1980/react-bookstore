# Bookstore - ReactJS
This project consist of building a simple Bookstore app using ReactJS and Redux

<img src="public/screenshot.png" >

## Technologies used

- ReactJS / Redux
- HTML
- CSS
- Git

## Usage

- git clone https://github.com/fegzycole/react-bookstore.git

- cd react-bookstore

- npm install

- npm start
